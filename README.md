# Short URL
A short url server with python + flask  
You can set some configuration in "config" file

## Before start service
You need to execute "pip install -r requirment.txt" to install flask module

## How to start service
Just execute with following command
python service.py

## How to access the service
The service port is set in config file, the default port is 5000.  
Open your browser with "http://127.0.0.1:5000"  
After get your short URL, open the short URL and it will redirect to original URL.
